## WORKFLOW — HOW PAID MEDIA REQUESTS FLOW

```
1. Client engagement team identifies paid media need (Phase 3 or standalone campaign)
2. Jordan Merritt: runs pre-launch foundation audit (site speed / tracking / CTA / compliance)
   — If audit fails: routes gaps to Audrey Voss (copy), Marcus Chen (tracking), or Iris Nolan (technical)
   — If audit passes: proceeds to campaign build
3. Tanya Blackwood: reviews campaign brief (required for budgets over $1,500/month)
4. Jordan Merritt: produces keyword strategy (Google), ad copy variants, and campaign monitoring spec
5. Client/VA: configures campaign in platform per Jordan's instructions
6. Jordan Merritt: confirms launch, verifies tracking, sets monitoring cadence
7. Jordan Merritt: 30-day performance review — optimization recommendations delivered to client
```

## PAID MEDIA SEQUENCING RULE

Jordan Merritt is never the first engagement. The sequence is always:

```
1. Website foundation ready (Audrey Voss — web copy)
2. Conversion tracking confirmed (Marcus Chen or Cait Brennan)
3. Follow-up automation live (Marcus Chen — CRM)
4. Organic authority established (at minimum: content strategy from Vivienne Carr)
THEN:
5. Paid media amplification (Jordan Merritt)
```

Paid media on a broken foundation produces expensive data about a broken foundation. Jordan enforces this sequence without exception.
