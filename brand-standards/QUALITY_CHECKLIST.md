# QUALITY_CHECKLIST.md
# Authority Systems Group™ — Mandatory QC Gate
# Every deliverable must pass all four gates before leaving ASG.

---

## GATE 1 — STRATEGIC ALIGNMENT
*Lens: Daniel Frost, CSO*

Ask these questions of every deliverable before approval:

- [ ] Does this deliverable serve the goals in `client-context.yaml`?
- [ ] Is the sequencing correct — is this the right move at the right time for this client?
- [ ] Are all recommendations specific to this client's stage, niche, and geography?
- [ ] Does the 1-year plan have a logical quarterly build (not 4 independent quarters)?
- [ ] Do the 3-year and 5-year plans follow logically from the 1-year foundation?
- [ ] Are all competitive claims accurate and specific to named competitors?
- [ ] Is there a clear "why this, why now" for every recommendation?

**FAIL STATE**: Generic strategy that could apply to any business in any market.

---

## GATE 2 — MESSAGING INTEGRITY
*Lens: Vivienne Carr, CMO*

Ask these questions of every content, copy, or messaging deliverable:

- [ ] Could any sentence in this document have been written for a different client?
  - If yes → rewrite it with specific client context
- [ ] Does the copy move through the Belief-to-Buy Framework™ (does it meet the avatar where they are, not where we want them to be)?
- [ ] Is the brand voice consistent with the client's approved voice guide?
- [ ] Are there any weasel words, vague claims, or generic "quality service" language?
  - If yes → replace with specific, outcomes-based language
- [ ] Does every hook earn the right to make its promise?
- [ ] Is there anything in here that a competitor could copy and use tomorrow?
  - If yes → make it more specific

**FAIL STATE**: Copy that sounds like it came from a template or could belong to any firm in the category.

---

## GATE 3 — REVENUE LOGIC
*Lens: Tanya Blackwood, CRO*

Ask these questions of every revenue-facing deliverable:

- [ ] Does every recommended strategy have a traceable path to revenue?
  - Name the path: strategy → action → lead/client → dollar amount
- [ ] Are revenue projections realistic (conservative scenario is achievable without extraordinary effort)?
- [ ] Are the Fast Cash strategies truly fast — executable within 30-90 days?
- [ ] Is the sequencing revenue-rational (highest-return strategies first)?
- [ ] Are there any "brand awareness" or "long-term play" recommendations without a short-term revenue anchor?
  - If yes → add the anchor or cut the recommendation
- [ ] Does the client know what to DO, not just what to think?

**FAIL STATE**: Impressive-sounding strategy with no clear revenue mechanism or implementation path.

---

## GATE 4 — DELIVERY STANDARD
*Lens: Preston Adler, COO*

Check every deliverable against these standards before export:

### Completeness
- [ ] All sections are complete — no placeholder text, no "[INSERT X]" remaining
- [ ] All client-specific data is accurately reflected (name, services, competitors, metrics)
- [ ] Table of contents matches actual document structure
- [ ] All cross-references are accurate

### Formatting
- [ ] Brand colors applied: #25aae1 (ASG Blue), #58585a (ASG Charcoal)
- [ ] Logo placement: top-left on all pages
- [ ] Footer: "Authority Systems Group™ — Confidential. Prepared exclusively for [Client Name]."
- [ ] Font hierarchy consistent throughout
- [ ] Page numbers present on all pages except cover
- [ ] Section headings use correct heading style (H1/H2/H3)

### Document Standards
- [ ] Cover page includes: client name, deliverable name, date, ASG branding
- [ ] Executive summary present and signed by appropriate persona
- [ ] Document named correctly: `[slug]_[deliverable-name]_[YYYYMMDD].docx`
- [ ] Saved to correct client output folder

### Final Check
- [ ] Read the first paragraph of every section — does it earn the reader's attention?
- [ ] Would Roger Bauer be comfortable putting his name on this document?
- [ ] Would a client who paid $5,000/month feel this was worth significantly more than they paid?

**FAIL STATE**: Incomplete sections, placeholder text, formatting inconsistencies, or any content that would embarrass ASG if seen by a prospect.
